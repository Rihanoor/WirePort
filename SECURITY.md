# Security Policy

## Supported Versions

Currently, only the latest version of WirePort is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

We take the security of WirePort seriously. If you find a security vulnerability, please do **not** report it via a public issue. Instead, report it privately by emailing us.

*   **Email:** rihanoor@lightib.com
*   **Response Time:** We will acknowledge receipt of your report within 48 hours and provide updates as we work to resolve the issue.

Please include:
1. A description of the vulnerability.
2. Steps to reproduce the issue (or a proof-of-concept script/config).
3. The potential impact of the vulnerability.
